To implement this solver, please 

[1] run startup.m first to add the path;

[2] run demonXXXX.m to solve different problems

This source code is programmed based on the algorithm described in:

Shenglong Zhou, Sparse SVM for Sufficient Data Reduction, 
IEEE Transactions on Pattern Analysis and Machine Intelligence, 10.1109/TPAMI.2021.3075339, 2021.

Please give credits to this paper if you use the code for your research.
